CSS : 캐스케이딩(상속/옷) 스타일 시트(Cascading Style Sheets)
    - HTML(뼈대) - HTTP
    - CSS(스타일-옷)
    - Javascript(행위/행동 - 이벤트)
        - CSS와 Javascript는 HTML에 포함된 구조

1. CSS 적용하는 방법
    1. CSS파일 만들어서 HTML <head>태그에 호출방식[*]
        <link rel="stylesheet" href="css파일경로">
    2. HTML <head>태그에 <style> 태그를 사용하는 방식
        <style></style>
    3. HTML <마크업 style=""> style속성을 이용하는 방식
        style=""

2. CSS 작성하는 방법
    어디의 {무엇을 : 어떻게;}

    1. 어디의[선택자]
        1. *        : 모든 곳에 적용
        2. 마크업   : 해당 마크업만 적용   
        3. class    : 해당 class속성에 동일한 이름을 갖는 마크업 적용
        4. id       : 해당 id속성에 동일한 이름을 갖는 마크업 적용
        * class명과 id의 이름은 아무거나[*권장: 영문,의미부여]
        <input class="titleInput" />
        <table id="boardTable"></table>

        *식별자 우선순위 [동일한 속성이 겹쳣을때]
            * < 마크업 < class < id

3. CSS 속성 종류
    1. background-color : 배경색
    2. background-image : url("이미지 경로")
        - background-repeat : 여백이 있을 경우 배경 반복 여부
            repeat[기본값], no-repeat

        - background-position : 배경이미지 위치
        
    ==== 글꼴 관련 ====
    2. font-family              : 폰트
        - 웹폰트[눈누 홈페이지 추천]
        - 예시
        @font-face { /* 웹폰트 */
            font-family: 'YClover-Bold';
            src: url('https://cdn.jsdelivr.net/gh/projectnoonnu/noonfonts_231029@1.1/YClover-Bold.woff2') format('woff2');
            font-weight: 700;
            font-style: normal;
        }
    3. font-size                : 폰트크기 [px:픽셀]
    4. font-style : italic;     : 기울기
    5. font-weight : bold;      : 진하게
    6. color                    : 색상
    7. word-spacing             : 어간(단어간격)[px]
    8. letter-spacing           : 자간(글자간격)[px]
    9. line-height              : 줄간격[%]
    10. text-align              : 텍스트 가로 정렬
        - left, right, center     : 왼쪽, 오른쪽, 가운데
    11. text-decoration         : 텍스트 선긋기
        - line-through 취소선
        - underline 밑줄
        - overline 윗줄
    12. text-shadow             : 텍스트 그림자
        - x축 y축 흐림정도 색상

    13. list-style-type         : 목록 사용시 글머리 타입

    ==== 박스 모델 관련 ====
    1. border
        : 두꼐 선종류 색상
        border-top, border-right, border-bottom, border-left
        : *선종류
        solid , dotted , double , dashed
    
    2. padding              : 안쪽 여백
        padding-top, padding-right , padding-bottom , padding-left

    3. margin               : 바깥 여백
        margin-top , margin-right , margin-bottom , padding-left

        -효율적
        padding : 10px;                 전체
        padding : 10px 20px;            상+하, 좌+우
        padding : 10px 20px 30px 40px;  시계방향 [위 오른쪽 아래 왼쪽]

        - 박스모델(block) 가운데 정렬
        margin : 0 auto;

        - 박스모델(inline) 가운데 정렬
        text-align :center;         - 텍스트 , img , a 등등

    4. box-sizing                   : 안쪽 여백과 테두리를 사이즈에 포함여부
        box-sizing : content-box;   : 포함X
        box-sizing : border-box;    : 포함O [*권장]

    5. width                        : 가로 사이즈

    6. height                       : 세로 사이즈

    7. display                      : 배치 방법
        display : inline;           : 한 줄에 포함된 배치
            마크업 : <span>, <img>, <a> 등
            - width, height 적용 불가

        display : block;            : 한 줄을 차지한 배치
            마크업 : <div>, <h>, <p>, <ul>, <ol>, <li> 등
            - width, height 적용 가능

        display : inline-block;     : 한 줄에 포함되고 사이즈가 적용된 배치
            - inline + block
            - width, height 적용 가능

        display : none;             : 표시안함.

        display : flex;             : 하위요소들 배치
            1. flex-wrap            : 하위요소 줄바꿈 여부
                nowrap              : (기본값)하위요소 줄바꿈X
                wrap                : 하위요소 줄바꿈O
                wrap-reverse        : 하위요소 줄바꿈O + 반대로

            2. flex-direction       : 하위요소들을 배치 방향
                row                 : (기본값)하위요소들을 가로배치
                column              : 하위요소들을 세로배치
                row-reverse         : 하위요소들을 가로배치 + 반대로
                column-reverse      : 하위요소들을 세로배치 + 반대로

            3. justify-content      : 하위요소들 가로 정렬
                flex-start          : (기본값)왼쪽부터 정렬
                flex-end            : 오른쪽부터 정렬
                center              : 가운데 정렬
                space-between       : 양쪽 끝에 여백없이 사이 여백 균등 정렬
                space-around        : 양쪽 끝에 여백은 절반으로 하고 사이 여백 균등 정렬
                space-evenly        : 양쪽 끝에 여백관 사이 여백 균등 정렬

            4. align-items          : 하위요소들 세로 정렬
                stretch             : (기본값) 부모의 높이만큼 자동으로 들어남.
                flex-start          : 윗변 정렬
                flex-end            : 밑변 정렬
                center              : 가운데 정렬
                baseline            : 내용물 아래 기준으로 정렬

    8. box-shadow                   :  박스모델 그림자
        - x축 y축 흐림정도 색상

    9. overflow                     : 하위 요소가 상위요소[고정사이즈]보다 커질때 
        visible[기본값], hidden, scroll, auto
